# 베타 종료 설문지 (구글폼 옮길 항목)

> 베타 종료 1주 전 발송, 마감일 종료일 = 정식 출시 마케팅 자료의 핵심.
> 구글폼·노션 폼 등으로 구현. 평균 작성 시간 5~7분 목표 (10분 넘으면 응답률 ↓).

---

## 📝 폼 헤더 (인사·동기 부여)

```

선생님께서 4주 동안 보내주신 피드백 덕분에
바로일지가 훨씬 더 좋은 도구가 됐습니다.

마지막으로 5~7분만 시간 부탁드립니다.
응답하신 분 모두에게 정식 출시 후 1년 무료 이용권 +
스타벅스 기프티콘 ☕ 보내드립니다.

전부 익명 처리되고, 솔직한 답변이 가장 도움 됩니다.
한 줄도 환영이고, 길게 써주셔도 감사드립니다!
```

## 📊 결과 분석 (베타 종료 후 5일 안에)

### 정량 지표
- **NPS** (Q8): 9~10 점 비율 - 0~6 점 비율 = NPS 점수
- **평균 만족도** (Q7)
- **평균 가격 수용선** (Q12): 중앙값
- **유료 전환 의향 비율** (Q14): "1~2개월 안에 결정" + "출시 첫 달 바로 가입" 비율
- **시간 단축 평균** (Q4)
- **추천 의향 비율** (Q17): "동료에게 카톡 소개" 이상 응답률

### 정성 인사이트
- **Q9 인용문 5개** → 출시 랜딩 페이지·영상 카피
- **Q10·Q11 페인포인트** → v1.1 우선순위
- **Q15 기능 요청** → 백로그 우선순위
- **Q18 마지막 한마디** → 운영자 동기부여 + 출시 자료 인용
