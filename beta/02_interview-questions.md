# 베타 1대1 인터뷰 가이드 (2주차, 30분 × 10명)

> 베타 2주차에 사용자 10명과 1대1 카톡 통화 또는 대면 인터뷰.
> 1인 운영 가능 사이즈 — 한 주에 5명 × 30분 = 2.5시간.
> 통화는 카톡 보이스톡 또는 통화 권장 (둘 다 안 되면 텍스트 인터뷰).

---

## 🎤 인터뷰 시작 (Opening, 2분)

```
선생님 안녕하세요. 시간 내주셔서 정말 감사합니다.

바로일지 베타 사용해보시면서 느끼신 점 솔직하게 들으려고
30분 정도 시간 부탁드렸습니다.

녹음은 안 하고, 메모만 할게요.
좋은 점·나쁜 점·이상한 점 다 환영입니다.
한 가지 부탁드릴 건 "괜찮아요" 보다는 진짜 느낌
말씀해주시면 큰 도움 됩니다 :)

그럼 천천히 시작해볼게요.
```

---

## 📋 질문 리스트 (10개 + 보조)

각 질문 후 후속 "왜요?" / "예를 들면?" 으로 깊이 파기.

### 🟦 첫인상 (2개)

**Q1. 처음 가입할 때 느낌 어떠셨어요?**
- 회원가입 폼이 길지 않았나요?
- 동의 화면이 부담스럽지 않았나요?
- (후속) 친구한테 추천할 때 어떻게 설명하시겠어요?

**Q2. 첫 화면(대시보드) 보고 어디부터 손이 가셨어요?**
- 무엇이 가장 먼저 눈에 들어왔어요?
- 헷갈리는 메뉴 있으셨어요?

### 🟦 핵심 흐름 (4개)

**Q3. 일정표 만들 때 막히신 곳 있었나요?**
- 아동 선택 → 요일·시간 입력 → 생성 흐름이 자연스러웠나요?
- 한 달치 자동 생성 — 시간 얼마나 단축됐다고 느끼세요?

**Q4. 기록지 작성 흐름은 어떠셨어요?**
- 미리 작성 + 월말 엑셀 자동 매칭 사용해보셨나요?
- 옛날 방식(엑셀·한컴)이랑 비교하면?
- (후속) 결과 입력만 하면 끝나는 게 진짜 그렇게 느껴졌나요?

**Q5. 한글파일 출력본 — 실제 점검·정산에 그대로 쓸 수 있는 수준인가요?**
- 양식이 평소 쓰는 거랑 같나요?
- 수정 안 해도 그대로 인쇄 가능하셨나요?
- (가장 중요) 지자체에 그대로 제출 가능할 것 같으세요?

**Q6. 승인내역 점검 기능 써보셨어요?**
- 엑셀 드래그 → 자동 진단 흐름이 명확하셨나요?
- 빨간색 알림 정확했나요? 잘못 잡힌 케이스 있었어요?

### 🟦 가치·우선순위 (2개)

**Q7. 가장 자주 쓰게 되는 기능 1개는 뭐예요?**
- 어떤 상황에서 가장 쓸모 있었어요?
- 이 기능 없으면 다시 옛날 방식 돌아가야 하는 정도예요?

**Q8. "이거 있었으면 좋겠다" 기능 1개 꼽으신다면?**
- 어떤 상황에 답답하셨어요?
- 우선순위로 만들어야 하는 정도예요? 있으면 좋은 정도예요?

### 🟦 출시 후 의향 (2개)

**Q9. 정식 출시 후 월 얼마면 계속 쓰시겠어요? 개인 의견 솔직히 부탁드려요.**
- (대답 듣고) 왜 그 가격이라고 느끼셨어요?
- 1만원 / 2만원 / 3만원 차이로 망설일 것 같으세요?

**Q10. 같이 일하는 동료 또는 다른 센터 선생님께 추천하실 의향이 있으세요?**
- 0~10점이면 몇 점 정도?
- (8점 이상) 어떻게 추천하실 것 같아요?
- (7점 이하) 무엇이 부족해서 그러세요?

## 🎤 보조 질문 (시간 남으면)

- 보호자께서 본 도구 사용에 대해 문의하신 적 있나요?
- 본 센터 운영하는 분이라면 자체 처리방침에 바로일지 추가하는 게 부담스러우신가요?
- 지자체 점검 받을 때 본 도구 출력물 제시한 적 있나요? (있으면 반응 어땠나요?)
- 다른 치료사 도구 (케어플센터·센터다이어리 등) 써보신 적 있으세요?

```
마지막으로 한 가지 — 정식 출시 후
선생님 같은 분이 후기 1줄 남겨주시면 큰 도움 됩니다.
부담 없으실 때 부탁드려도 될까요?

오늘 시간 내주셔서 정말 감사합니다.
앞으로도 어려운 점 있으시면 단톡방·카톡으로 편하게 연락 주세요.
```

## 📊 인터뷰 10명 종합 후 작성할 보고서 (1페이지)

베타 종료 1주 전 모든 인터뷰 마치고 작성:

```
1. 핵심 인사이트 Top 3 (10명 중 5명 이상 언급)
2. 가장 사랑받은 기능 (실명 인용 5개)
3. 가장 자주 호소된 페인 (수정 우선순위 정렬)
4. 평균 NPS·만족도
5. 평균 가격 수용선 (월 단가 의견 평균·중앙값)
6. 출시 마케팅에 쓸 인용문 5개 (보고자 허락 받은 것)
7. v1.1 로드맵 우선순위 3개
```

→ 이 보고서가 **정식 출시 마케팅 자료 + 가격 결정 + v1.1 우선순위**의 근거가 됨.
