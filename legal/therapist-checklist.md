# 「바로일지」 도입 시 치료사용 체크리스트

> 본 문서는 발달재활 치료사(센터 운영자)가 「바로일지」를 본인의 업무에 도입할 때 따라야 할 행정·법적 체크리스트입니다. 회기 운영의 안전성과 점검 대비를 위해 권장됩니다.

---

## 1. 정부 표준 동의 확인 (대부분 이미 완료)

자녀가 발달재활서비스를 신청하기 위해 보호자께서 지자체에 제출하신 동의서를 한 번 확인하세요. **새로 받을 필요는 없습니다.**

| 확인 항목 | 상태 |
|---|---|
| **서식 1-2호** 법정대리인 동의서 (국민행복카드용) | □ 확인 |
| **서식 1-3호** 상담전화 개인정보 제공동의서 | □ 확인 |
| **서식 1-4호** 개인정보 수집·이용 및 제3자 제공 동의서 ⭐ | □ 확인 |

→ 위 동의서는 정부(사회서비스 전자바우처 시스템)가 보관하므로, 치료사는 직접 보관할 필요 없음. 다만 사본 1부를 본 센터에 비치하면 점검 시 빠릅니다.

## 2. 본 센터의 개인정보처리방침에 「바로일지」 추가

본인 센터의 자체 개인정보처리방침이 있다면 (없다면 이번 기회에 작성 권장), 다음 내용을 **처리 위탁 항목**에 추가하세요.

### 추가 문구 예시 (그대로 복붙 가능)

> ※ 본 센터는 회기 일정·기록 관리 업무를 다음 수탁자에게 위탁하고 있습니다.
> 
> | 수탁자 | 위탁 업무 | 보유 기간 |
> |---|---|---|
> | **바로일지** (https://baroilji.com) | 회기 일정·기록 클라우드 저장 및 한글파일 자동 생성 | 위탁계약 종료 시까지 |
> | 서버 위치 | 대한민국 국내 (네이버 클라우드, 춘천) | — |

→ 이 한 표만 본 센터 처리방침에 추가하면 「개인정보 보호법」 제26조 (처리 위탁 공개) 요구사항 충족.

## 3. 보호자 안내문 비치

`parent-notice.md` (보호자용 안내문) 을 인쇄해두세요.

| 행동 | 시점 |
|---|---|
| 첫 회기 시 보호자께 1부 전달 | 회기 시작 시 |
| 본 센터에 비치 (점검 대비) | 상시 |
| 보호자 문의 시 즉시 제공 가능 | 필요 시 |

별도 서명은 필요하지 않으며, **"안내했다"는 사실** 자체가 중요합니다 (= 처리 위탁 사실의 공개·고지).

## 4. 회기 기록·영상 촬영 동의서 (기존 운영) 점검

「바로일지」 도입과 무관하게, 다음 자체 동의서들은 본 센터에서 별도로 운영해야 합니다.

| 동의서 | 무엇 | 의무성 |
|---|---|---|
| 서비스 이용 계약서 | 회당 단가, 결제, 환불 등 | 권장 |
| 회기 영상·녹음 동의서 | 회기 중 녹화·녹음 시 | **필수** (촬영 시) |
| 보호자 상담 기록 동의서 | 상담 내용 기록·공유 | 권장 |

→ 본인 센터의 기존 양식을 그대로 사용하면 됩니다. 「바로일지」가 대체하지 않습니다.

## 5. 보안·관리 행동 수칙

「바로일지」 사용 시 다음 사항을 지켜주세요.

### 계정 보안

- [ ] 비밀번호는 8자 이상, 영문·숫자·특수문자 조합
- [ ] 본인 계정을 다른 사람과 **절대 공유하지 않기** (1인 1계정)
- [ ] 공용 PC에서 사용 시 반드시 **로그아웃** 후 자리 비우기
- [ ] 의심스러운 접근 발견 시 즉시 비밀번호 변경

### 정보 입력 시

- [ ] 정부 동의서(서식 1-4호) 범위 안의 정보만 입력
- [ ] 회기 결과 기록에 **민감한 진단명·약물 정보** 등 작성 시 신중히 표현
- [ ] 다른 아동의 이름·정보가 섞이지 않도록 주의

### 한글파일 출력 후

- [ ] 출력된 일정표·기록지의 정확성 **본인 검토** 의무
- [ ] 인쇄 후 종이는 잠금 캐비닛에 보관, 5년 후 분쇄 폐기
- [ ] PDF 등으로 외부 공유 시 보호자 동의 확인
