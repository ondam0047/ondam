## 🎯 발달재활 특수성 반영 부분

일반 SaaS 약관과 다른 점 — 이 부분이 변호사 검토에서 가장 중요해요.

### 1. 만 14세 미만 아동 정보 처리
- 치료사가 사용자, 아동이 정보 주체 → 처리수탁자 지위
- 법정대리인(보호자) 동의 확보 책임은 치료사에게 위임
- 회사는 안전한 저장·관리 기술 제공자 지위
- → `privacy-policy.md` 제2조 4항 / `terms-of-service.md` 제12조

### 2. 한글파일 자동 생성 결과물 면책
- 일정표·기록지는 회원 입력 기반 자동 생성물
- 최종 사용 전 회원 검토 의무
- 부정확성으로 인한 지자체 점검 사고 면책
- → `terms-of-service.md` 제13조

### 3. 승인내역 점검 도구의 한계
- 보조 도구이지 공식 점검 대체 효력 없음
- → `terms-of-service.md` 제13조 3항

### 4. 의료 자문이 아님
- 본 서비스는 **업무 보조 도구**
- 의료법 적용 대상 아님 명시
- → `terms-of-service.md` 제22조
