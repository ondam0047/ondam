# 참고 — 정부 표준 양식·법령 URL 카탈로그

> 발달재활서비스 운영 시 참고할 정부 표준 양식과 관련 법령의 공식 출처 모음입니다. 직접 다운로드는 정부 사이트 봇 차단 정책으로 자동화가 어렵고, **사람이 브라우저로 접속**해서 받으셔야 합니다.

## 1. 보호자가 지자체에 제출하는 표준 동의서 (발달재활서비스 신청 단계)

| 서식 | 명칭 | 다운로드 |
|---|---|---|
| **서식 1-2호** | 사회서비스 전용 국민행복카드 발급을 위한 법정대리인 동의서 | [정부24](https://www.gov.kr/portal/service/serviceInfo/VOU000000050) |
| **서식 1-3호** | 국민행복카드 상담전화를 위한 개인정보 제공동의서 | 위 정부24 패키지에 포함 |
| **서식 1-4호** ⭐ | 개인정보 수집·이용 및 제3자 제공 동의서 | 위 정부24 패키지에 포함 |
| **서식 4-1호** | 발달재활서비스 의뢰서 (9세 미만 장애 미등록 영유아용) | [창원시 다운로드](https://www.changwon.go.kr/cwportal/cmsfile/download.do?idx=223674&fsiz=56171) |
| **별지 제9호** | 개인정보 수집·이용 동의서 (장애아동복지지원법 시행규칙) | [법제처](https://law.go.kr/flDownload.do?flSeq=42483175) |

### 핵심 — 서식 1-4호의 의미

서식 1-4호는 보호자가 **사회서비스 전자바우처 시스템 가입 시 1회 동의**하는 문서입니다. 이 동의 범위에 **"서비스 제공기관에 정보 제공"** 항목이 포함되어 있어, 치료사가 본인 업무 목적으로 회기 기록을 작성하는 행위는 이미 동의된 범위 안입니다.

→ 「바로일지」 사용에 따른 별도 보호자 동의는 법적으로 필수가 아님 (안내·고지로 충분).

## 5. 다운로드 가이드 (브라우저로 받는 법)

정부 사이트는 자동 다운로드(curl·wget)를 차단합니다. 브라우저로:

1. 위 URL 클릭 → 정부 페이지 열림
2. 페이지 안에서 첨부 파일 (.hwp / .hwpx / .pdf) 클릭하여 저장
3. 한컴오피스 또는 한컴 뷰어로 열기

`legal/references/` 폴더에 다운받은 파일 추가 시 `.gitignore` 처리 권장 (저작권 이슈 회피).
