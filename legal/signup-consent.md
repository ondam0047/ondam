# 회원가입 동의 화면 텍스트

> 회원가입 페이지에 표시할 동의 항목 텍스트입니다. 회원이 가입 시 체크박스로 동의를 받습니다.
>
> 본 서비스는 **발달재활 치료사**(사용자)를 위한 업무 보조 도구로, 아동·보호자 정보는 정부 표준 동의 체계(서식 1-4호)에 따라 이미 확보된 것을 전제로 합니다.

## 동의 항목 상세

### 필수 동의 1 — 만 14세 이상 확인

> 만 14세 이상입니다.

**동의 거부 시**: 가입 불가 (법정대리인 동의 처리 시스템 미구비)

---

### 필수 동의 2 — 서비스 이용약관

> 바로일지 서비스 이용약관에 동의합니다.

**클릭 시 표시**: [이용약관 전문](./terms-of-service.md)

**핵심 안내**:
- 본인 계정은 본인만 사용 (공유 금지)
- 한글파일 자동 생성물의 정확성은 본인이 검토
- 한 번의 손해배상 한도는 12개월 이용요금 이내

---

### 필수 동의 3 — 개인정보처리방침

> 개인정보처리방침에 동의합니다.

**클릭 시 표시**: [개인정보처리방침 전문](./privacy-policy.md)

**핵심 안내**:
- 이메일, 이름, 치료사 종류 등 수집 (회원 본인 정보)
- 국내(춘천) 서버에만 저장, 국외 이전 없음
- 회사는 회원이 입력하는 아동 정보에 대하여 **처리 수탁자** 지위
- 회원 탈퇴 시 30일 이내 파기

---

### 필수 동의 4 — 발달재활 자격 확인 및 보호자 안내 의무

> 본인은 발달재활서비스 제공인력으로 등록·종사 중이며, 본 서비스 사용 사실을 보호자에게 안내할 의무를 이해합니다.

**클릭 시 표시 (안내 모달)**:

```

[법적 구조]
보호자께서는 사회서비스 전자바우처 가입 시
「개인정보 수집·이용 및 제3자 제공 동의서」(서식 1-4호)에
이미 동의하셨으며, 이 동의 범위에는 "서비스 제공기관(센터)에
정보 제공"이 포함됩니다.

본인(치료사)이 회기 기록 작성을 「바로일지」에 위탁하는 것은

별도의 보호자 서면 동의가 필요하지 않습니다.
다만 처리 위탁 사실의 **공개·고지**는 본인의 의무입니다.

[다음 사항을 확인하셨나요?]

✓ 본인은 「장애아동복지지원법」 등 관련 법령에 따라
  발달재활서비스 제공인력으로 등록·종사 중
✓ 본 센터의 자체 개인정보처리방침에 「바로일지」를
  처리 수탁자로 추가할 예정 (또는 이미 추가됨)
✓ 첫 회기 시 보호자께 「바로일지」 사용 안내문
  (parent-notice) 1부를 전달

→ 가이드: [치료사용 체크리스트 다운로드]
→ 보호자 안내문 양식: [parent-notice 다운로드]
```

**동의 거부 시**: 가입 불가

### 선택 동의 5 — 마케팅 정보 수신

> 마케팅 정보 수신에 동의합니다.

**클릭 시 표시**: [마케팅 동의 전문](./marketing-consent.md)

**거부 시**: 가입 가능. 보안·약관 변경 등 운영상 필수 안내는 계속 발송됨.

## UI/UX 구현 메모

- 전체 동의 체크박스는 상단/하단에 모두 배치 (편의성)
- "필수" 동의 미체크 시 [가입하기] 버튼 비활성화
- 약관 전문은 별도 페이지 또는 모달로 표시
- 약관 페이지에는 **시행일자, 변경 이력** 명시
- 동의 일시는 DB 에 저장 (감사 기록용)
- 필수 동의 4의 모달에서 **체크리스트·안내문 다운로드 링크** 노출 → 가입 직후 활용 유도

### 동의 기록 DB 스키마 추가 (개발 시)

```sql
-- 가입 시 동의 기록 보관
CREATE TABLE user_consent (
  id           SERIAL PRIMARY KEY,
  user_id      INTEGER NOT NULL REFERENCES user(id),
  terms_v      TEXT NOT NULL,     -- 약관 버전 (예: "v1.0_20260601")
  privacy_v    TEXT NOT NULL,     -- 개인정보처리방침 버전
  professional BOOLEAN NOT NULL,  -- 필수 동의 4 (자격·안내 의무)
  marketing    BOOLEAN DEFAULT false,
  consented_at TIMESTAMP DEFAULT NOW(),
  ip_address   TEXT,
  user_agent   TEXT
);
```
