## 🎯 출시 가능 조건 (Go/No-Go 체크)

출시 D-day에 다음 9개 다 만족하면 GO:

```
🔴 사업자등록·통신판매업 신고 완료
🔴 K-Startup 심화 상담 자문서 1건 이상 수령·반영
🔴 약관·처리방침·환불정책 사이트 게시 (TBD 모두 채워짐)
🔴 회원가입 동의 체크박스 + user_consent DB 가동
🔴 결제·환불 흐름 종합 테스트 통과
🔴 운영자 본인 2FA 설정 완료
🔴 푸터 사업자 정보 + 약관 링크 게시
🔴 베타 사용자 1명 이상 정식 회원 전환 검증
🔴 사고 대응 매뉴얼 1장 + 보호자 응대 채널 가동
```

→ 9개 중 1개라도 미달 = 출시 1주 연기 권장.
