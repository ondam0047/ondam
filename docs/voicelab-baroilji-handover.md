## 🚫 바로일지에 통합 X (VoiceLab 학과용으로만 유지)

다음 모듈은 의료기기 분류 위험이 높아 **바로일지에 통합 안 함**:

| 위험 모듈 | 위험 이유 |
|---|---|
| `/diagnosis` 경로 | "진단" = 의료기기법 핵심 트리거 |
| **PD Voice Diagnosis** | 파킨슨병(Parkinson's Disease) 진단 |
| **VHI** (음성장애지수) | 표준 임상 평가 도구 (Jacobson 1997) |
| **지산학 평가** (Cog·Lang·Mood·QoL) | 인지·정서·삶의 질 임상 평가 |
| **음질 분석** (jitter·shimmer·HNR) | 임상 음성 평가 지표 |
| **자발화 분석** | 임상 평가 도구 |
| 모음·자음 시각화 3D | (Pro로 단순화 시 가능, 일단 제외) |
| 페이스 조절 훈련 (원형) | (단순화한 "말속도 조절 연습"으로 통합) |

→ 위 모듈들은 VoiceLab(학과용)에서만 운영. 학과 교육·연구 환경에서는 의료기기 분류 위험 X.
