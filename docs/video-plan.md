# 바로일지 사용법 영상 구성안 (베타 온보딩용)

> 권장: **짧은 영상 6편 + (옵션) 30초 인트로 1편.** 한 편당 1~3분, 화면 녹화 + 자막/말풍선.
> 촬영 순서가 곧 데이터 흐름이라, **1→6 순서대로** 찍으면 데모 데이터가 자연스럽게 쌓입니다.

## 촬영 전 준비
- 데모 계정 1개(가입 시 **초대코드** 필요 — `BETA_ACCESS_CODE`)
- 더미파일 2종(첨부):
  - `아동등록_샘플.xlsx` — 가상 아동 5명(김하늘·이바다·박가온·정아라·최도윤)
  - `서비스제공내역_샘플.xlsx` — 위 5명의 2026년 6월 회기(전자바우처 모사, **소급 1건·결제시간 겹침 1쌍** 포함)
- 가능하면 화면 1920×1080, 글자 크게(데모 계정 브라우저 확대 110~125%)

## 🎬 영상 1 — 시작하기 (가입·내 설정)  ⏱️1~2분
- **목적**: 가입하고 내 정보(치료사 종류·시간대·단가)를 맞추면 이후가 자동으로 채워진다.
- **흐름**: 가입(초대코드·이름·치료사 종류·이메일) → 로그인 → **[내 설정]**에서 치료사 종류·회기 시간대·기본 단가 확인/수정 → 저장.
- **멘트**: "치료사 종류만 고르면 서비스 종류·담당이 자동으로 본인으로 잡혀요. 시간대·단가는 한 번만 맞춰두면 끝."

## 🎬 영상 2 — 내 아동 등록 (한 명씩 + 엑셀 일괄)  ⏱️2~3분  📎`아동등록_샘플.xlsx`
- **목적**: 아동을 등록해두면 매월 자동 호출된다. 두 가지 방법.
- **흐름**:
  1. **한 명씩**: [내 아동]→[한 명씩 등록] → 이름·생년월일·기본 시간대·반복 요일·단가·본인부담금 입력(서비스·담당은 자동 고정) → 요일 고르면 **요일별 시간**도 조정 가능 → 등록.
  2. **엑셀 일괄**: [내 아동]→[엑셀로 가져오기] → `아동등록_샘플.xlsx` 끌어다 놓기 → 미리보기 → **[이대로 저장]**(5명 한 번에).
- **멘트**: "쓰던 명단이나 전자바우처 파일을 그대로 올리면 5명이 한 번에 등록돼요. 담당은 전부 본인으로."

## 🎬 영상 3 — 일정표 만들기 (개별 + 이번 달 일괄)  ⏱️2~3분
- **목적**: 한 달치 회기를 자동 생성하고 한글파일로 출력.
- **흐름**:
  1. **개별**: [일정표]에서 아동 불러오기 → 반복요일·시간 확인 → **[일정표 생성]** → 미리보기 → 저장 → **한글파일(.hwpx) 다운로드**.
  2. **이번 달 일괄**: 상단 **[이번 달 일괄 생성]** → 대상 월 선택 → 생성 → "새로 5건" 결과 확인.
- **멘트**: "한 명씩도 되고, 담당 아동 전체를 이번 달 한 번에도 만들 수 있어요. 공휴일은 자동 제외."

## 🎬 영상 4 — 기록지 작성 (직접 시작 + 엑셀 자동완성)  ⏱️3분  📎`서비스제공내역_샘플.xlsx`
- **목적**: 미리 작성도, 월말 엑셀로 일괄 자동완성도 된다.
- **흐름**:
  1. **엑셀 없이 직접 시작**: [기록지]→아동·월 선택→[작성 시작] → 일정표 회기가 자동 시드 → 결과 입력.
  2. **엑셀 자동완성**: `서비스제공내역_샘플.xlsx` 업로드 → 아동별 회기·승인번호·결제일 자동 채움 → **소급결제 N건** 배너 확인.
  3. (있으면) **[전월 기록 가져오기]** 로 지난달 총평 복사 → 한글파일 다운로드.
- **멘트**: "전자바우처 파일만 올리면 회기·승인번호가 알아서 들어가요. 소급결제도 빨간 배너로 알려줘요."

## 🎬 영상 5 — 승인내역 점검 (겹침·소급)  ⏱️1~2분  📎`서비스제공내역_샘플.xlsx`
- **목적**: 지자체 점검 전에 결제시간 겹침·소급결제를 미리 잡는다.
- **흐름**: [승인내역 점검] → `서비스제공내역_샘플.xlsx` 업로드 → **6/10 빨간색 행(결제시간 16분 간격=겹침)** 과 **소급 1건** 확인.
- **멘트**: "결제 시간이 너무 붙어 있으면 빨갛게 표시돼요. 환수 위험을 미리 점검."

## 🎬 영상 6 — 일괄 다운로드 & 내 시간표(출석부)  ⏱️1~2분
- **목적**: 만든 일정표·기록지를 한 번에 받고, 시간표·출석부도 확인.
- **흐름**: [일괄 다운로드] → 월·대상 선택 → 일정표/기록지 zip 다운로드. [내 시간표] → 월 회기 현황 + **출석부 엑셀** 다운로드.
- **멘트**: "한 달 작업을 한 번에 받고, 출석부 엑셀까지."
